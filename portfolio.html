Weather Dashboard

Weather Dashboard is a simple and easy to use applicaton to check the daily weather by city. The features include the actual date,name of the city, temperature, humidity, wind speed,and a 5 days forecast. This weather app supports global locations and definitely keeps the user updated with weather conditions.

WORK DAY PLANNER

Daily Planner is a project that was created to provide time management through organazing all the dayly events according to business hours from 10:00am to 17:00 pm. This planner also has the ability to display different colors to make a difference between past, current, and future events. All these events will be saved by using the save button making every day tasks easy to remember.
