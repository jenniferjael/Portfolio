CLASSY MOVIES is a team project and a friendly user application to search for classic movies. Classy Movies was built with the motivation of helping users to find those movies that bring good memories. This responsive application offers movies that can be searched by title or by gnere. One special feature is the ability to access the movie trailers by one simple click on the movie.

WEATHER DASHBOARD is a simple and easy to use applicaton to check the daily weather by city name. The features include the actual date,city name, temperature, humidity, wind speed, and a 5 days forecast. This weather app supports global locations and definitely keeps the user updated with useful information about his/her daily weather conditions.

DAILY PLANNER: This project was created to provide users with a daily planner where they can add events according to business hours from 10:00am to 17:00 pm. This planner also has the ability to display different colors according to past,present, and future events. If the event is set on the current hour, the textarea will display a red color, if the events were scheduled for hours before the current hour, the textarea will be display a gray color to indicate the past. Finnally, for events scheduled for after the current hours, the textarea will become green.
